# Migrate Google places between Google Accounts

* Go to https://www.google.com/bookmarks/ and press `Export Bookmarks`
* Add the path to the GoogleBookmarks.html file to constant `googleBooksmarksPath`
* Install dependencies `npm install`
* Launch `node main.js`
* Log in to the Google Account you want to import the places to
* Click the terminal and any button to continue the program
* Check the output png files to make sure your places are added to your new Google Account
